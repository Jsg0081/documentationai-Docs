---
title: Editor Not Loading
description: Resolve issues when the editor does not appear.
---

## Symptoms

- Blank screen where the editor should show
- Editor stuck on loading spinner

## Causes

- Browser cache issue
- Unsupported browser version

## Quick Fix

<Steps>
  <Step title="Clear your cache">
    ````
    Press Ctrl+Shift+R (or Cmd+Shift+R on Mac) to force reload.
    ````
  </Step>
  <Step title="Check browser support">
    Make sure you use the latest Chrome, Firefox, or Edge.
  </Step>
  <Step title="Try incognito mode">
    Disable extensions and open Snappytb in a private window.
  </Step>
</Steps>
