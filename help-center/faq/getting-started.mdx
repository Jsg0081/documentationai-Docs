## Overview

To get started with Snappytb, create your first project workspace.

## Steps

1. Login at `https://app.snappytb.com`
2. Click "New Project" at the dashboard
3. Enter a project name and description
4. Click "Create"

Your new workspace appears in the project list.

## Next

Select your project to start adding docs or invite collaborators.
